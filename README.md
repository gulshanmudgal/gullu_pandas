# Pandas Learning Project

This repository contains a Jupyter notebook demonstrating basic data exploration and manipulation using Pandas on the Stack Overflow Developer Survey dataset.

## Overview

The project includes:
- `00_Pandas_Demo.ipynb`: A Jupyter notebook with step-by-step Pandas operations
- `data/`: Folder containing the survey data files
  - `survey_results_public.csv`: Main survey responses
  - `survey_results_schema.csv`: Column descriptions and schema
- `notes/`: Additional documentation and notes
  - `pandas_demo_notes.md`: Detailed explanations of the notebook content

## Prerequisites

- Python 3.x
- Jupyter Notebook or JupyterLab
- Pandas library (`pip install pandas`)

## Getting Started

1. Clone or download this repository
2. Navigate to the project directory
3. Install dependencies: `pip install pandas`
4. Launch Jupyter: `jupyter notebook` or `jupyter lab`
5. Open `00_Pandas_Demo.ipynb` and run the cells

## Dataset

The data comes from the Stack Overflow Developer Survey, which collects insights about developers' experiences, technologies, salaries, and more. The survey is conducted annually to understand trends in the developer community.

## Notes

For detailed explanations of each step in the notebook, refer to `notes/pandas_demo_notes.md`.
